# Tutorial Penggunaan: Role Superuser

Role Superuser memiliki otorisasi penuh atas sistem. Berikut adalah panduan fungsinya.

## 1. Mengelola Pengguna

## 2. Fitur "Login Sebagai User Lain"
- Digunakan untuk melihat tampilan dashboard user tertentu tanpa perlu tahu password mereka.
- Pilih user dari dropdown, lalu dashboard akan otomatis berubah.
- Sangat berguna untuk bantuan teknis kepada warga.
